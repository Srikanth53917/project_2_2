# Stock Market Prediction with Sentiment Analysis

This project predicts stock market trends using historical stock prices
and sentiment analysis of news headlines.

## Features
- Historical stock price analysis
- Year-wise sentiment analysis using dataset
- Buy / Sell / Hold decision

## Technologies Used
- Python
- Machine Learning (Linear Regression)
- NLP (VADER Sentiment Analysis)
- Pandas, Matplotlib

## How to Run
1. Install required libraries:
   pip install -r requirements.txt
2. Run the project:
   python app.py

## Output
- Stock price trend graph
- Sentiment analysis bar graph
- Final prediction decision
